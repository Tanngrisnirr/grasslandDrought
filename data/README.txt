GRASSLAND DROUGHT DATA FOLDER
=============================

Place your Excel file (.xlsx) here.

The project will automatically detect and load it when you open RStudio.

Requirements:
- One Excel file in this folder
- Columns for: treatment, block, date, ID (auto-detected)
- Response variable (e.g., RootGrowth, ANPP, BNPP)

Example datasets:
- Picon-Cochard et al. (2021): https://zenodo.org/records/4034903
